# Process Creation using fork()

## Description
This program demonstrates process creation in Linux using the `fork()` system call. It creates a child process and displays the process IDs (PID) of both the parent and child processes.

## Concepts
- Process creation
- Parent and child processes
- Process IDs (PID)
- `fork()`
- `getpid()`
- `getppid()`

## Output
Displays the IDs of the parent and child processes.
